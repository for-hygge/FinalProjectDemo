# Order & Payment System (UI)

This project is a simple Order and Payment Management System UI built with React.

## Features
Order Management
- Create orders
- Query orders by user ID
- View order details
- View order payment information
- Cancel orders

Payment Management
- Create payment
- Query payment details
- Cancel payment
- Refund payment

## Project Structure
```
src/
├── assets/         # Images
├── components/     # Reusable UI components
├── data/           # Mock product data
├── pages/          # Page-level components (Home, Order, Payment)
├── services/       # API request functions
├── styles/         # Global style definitions
├── App.jsx         # Main application entry
└── main.jsx        # React entry point
```

## Getting Started
1. Install dependencies \
```npm install```

2. Run the frontend \
```npm run dev```

3. The app will be available at: http://localhost:5173/

## Notes
This project uses mock product data for demonstration. The UI is intentionally simple and focuses on functionality.
